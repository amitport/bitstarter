Tousands of card games have been invented and played since the 9th century. Some of the games became very popular and remained so for many years before being forgotten. Some of today's popular games have changed little over several hundred years.

An insanly large number of books, web sites, forums and wikis have been dedicated to the research of existing card games and the invention of new ones.

CARDSFOREST is a new type of tool for today's card game enthusiests.
Wiki
Collaboratively manage information about card games.
Development environment
Actually create those games.
Gaming platform
Then play them everywhere.
What technologies are you currently using?
The Wiki and the development environment are web based, using HTML5 and running on nodeJS.
The games are specified using Markdown and CoffeeScript and run on mobile devices via Apache Cordova.
